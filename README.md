## Features

- Home page:

  - Hero section
  - Products cards section
    - Product picture
    - Product name
    - Product price

- All Products page

  - Product card
  - product picture
  - product price

- Detail product Page

  - SKU ( Stock Keeping Unit )
  - product name, price, picture
  - description
  - add to cart button
  - input quantity ( button add to cart )

### Shoping Cart Page

- Cart produtcs item to buy
- product name, price, quantity, total
- remove product item
- Continue shopping button
- Link to checkout

### Checkout Page

- Order summary
  - Ordered product list
  - Delivery option
  - Payment option
  - Order summary
  - Create order button

### Login Page:

- Email
- Password
- Login Button
- Register Button

### Register Page:

- Name
- Email
- Password
- Address (optional)
- Phone (optional)
